# Download Page Assets

This folder contains images for the AI Video Editor download page.

## Required Images

Please add the following images to complete the download page:

### Logo & Branding

- `favicon.png` (32x32 or 64x64) - Browser tab icon
- `logo.png` (256x256 or larger) - Full logo

### Social Media Cards

- `og-image.png` (1200x630) - Open Graph image for social sharing
- `twitter-card.png` (1200x675) - Twitter card image

### Screenshots Folder

Add screenshots of your application showing:

- Main interface with video upload
- AI processing in action
- Generated clips view
- Settings panel

### Icons Folder

Platform-specific icons (optional, as SVGs are used in HTML):

- `windows-icon.png`
- `macos-icon.png`
- `linux-icon.png`

## Image Optimization Tips

1. **Compress images** using TinyPNG or similar tools
2. **Use WebP format** for better compression (with PNG fallbacks)
3. **Provide 2x versions** for retina displays
4. **Keep file sizes small** (under 500KB per image)
